# EN2091 Laboratory Practice and Project - 3 Channel Audio Mixer with 3 Band Equalizer 

Designed and implemented a compact analog audio mixer featuring three independent input channels, each with a 3-band equalizer (Bass, Mid, Treble) for precise tonal control.
